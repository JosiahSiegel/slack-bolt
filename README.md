Tips:

- Check vulnerabilities: `npm audit`
- Fix Glitch out of sync with repo:
  - `git pull`
  - `refresh`
- Hard refresh Glitch from repo:
  - `git fetch --all`
  - `git reset --hard origin/main`
  - `refresh`
